# Zepto Retail Inventory Analysis

## Project Overview

This project analyzes a Zepto retail inventory dataset containing 3,700+ product records using MySQL. The objective was to perform data cleaning, exploratory data analysis (EDA), and business-focused analysis to generate insights related to pricing, discounts, inventory, and product categories.

## Tools Used

* MySQL
* SQL
* CSV Dataset

## Key Tasks Performed

* Created and managed database tables.
* Performed data cleaning by removing invalid records.
* Converted product prices from paise to rupees.
* Identified duplicate products and missing values.
* Analyzed stock availability and category distribution.
* Calculated category-wise revenue and inventory weight.
* Identified highly discounted and out-of-stock products.
* Evaluated product value using price-per-gram analysis.

## Key SQL Concepts Used

* Aggregate Functions (SUM, AVG, COUNT)
* GROUP BY and HAVING
* CASE Statements
* Filtering and Sorting
* Data Cleaning Queries
* Exploratory Data Analysis (EDA)

## Results

Generated actionable insights on inventory management, discount trends, revenue estimation, and product pricing strategies.
